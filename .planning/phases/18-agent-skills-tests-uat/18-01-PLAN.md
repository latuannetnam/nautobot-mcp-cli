# Plan 18-01: Agent Skill File Rewrites

## Objective

Rewrite all 3 agent skills (`cms-device-audit`, `onboard-router-config`, `verify-compliance`) to reference the new 3-tool API (`nautobot_api_catalog`, `nautobot_call_nautobot`, `nautobot_run_workflow`). Consolidate multi-step patterns into single `nautobot_run_workflow()` calls where the workflow already handles data retrieval + comparison. Each skill starts with a discovery step and includes trimmed example response envelopes.

## Tasks

**Step 3: Compare Static Routes Against CMS Records (CONSOLIDATED)**
Merge old Steps 4+5 into single call:
```

  workflow_id="compare_routes",
  params={"device_name": "<device>", "live_routes": [<list from jmcp>]}
)
```

Required params: `device_name` (str), `live_routes` (list of dicts).

**jmcp steps:** Keep `execute_junos_command` calls identical. Add brief format notes for expected data shapes (see format notes above for `live_neighbors` and `live_routes`).
