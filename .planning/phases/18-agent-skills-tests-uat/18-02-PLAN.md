# Plan 18-02: UAT Tests & Live Server Validation

## Objective

Create UAT smoke tests in two forms: (1) pytest tests with `@pytest.mark.live` marker for CI-friendly execution, and (2) a standalone `scripts/uat_smoke_test.py` script for quick manual runs. Both validate the 3-tool API Bridge against the live Nautobot dev server at `http://101.96.85.93` (configurable via `NAUTOBOT_UAT_URL` env var). Uses `HQV-PE-TestFake` as the CMS test device.
