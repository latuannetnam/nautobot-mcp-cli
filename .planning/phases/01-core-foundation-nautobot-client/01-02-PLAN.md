1. Create `nautobot_mcp/models/device.py`:
   - `DeviceSummary(BaseModel)`:
     - `id: str, name: str, status: str`
     - `device_type: RelatedObject` (inline summary)
     - `location: RelatedObject`
     - `tenant: Optional[RelatedObject] = None`
     - `role: Optional[RelatedObject] = None`
     - `platform: Optional[str] = None`
     - `serial: Optional[str] = None`
     - `primary_ip: Optional[str] = None`
     - `@classmethod from_nautobot(cls, record) -> DeviceSummary`

2. Create `nautobot_mcp/devices.py`:
   - `list_devices(client: NautobotClient, location: str = None, tenant: str = None, role: str = None, platform: str = None, q: str = None, limit: int = 0, **extra_filters) -> ListResponse[DeviceSummary]`
     - Uses `client.api.dcim.devices.filter(**filters)` or `.all()` if no filters
     - Applies limit if > 0
     - Returns `ListResponse(count=len(all_results), results=limited_results)`
   - `get_device(client: NautobotClient, name: str = None, id: str = None) -> DeviceSummary`
     - Uses `.get(name=name)` or `.get(id=id)`
     - Raises `NautobotNotFoundError` if None returned
   - `create_device(client: NautobotClient, name: str, device_type: str, location: str, role: str, status: str = "Active", **kwargs) -> DeviceSummary`
     - Resolves device_type, location, role by name to UUIDs
     - Uses `.create(**data)`
   - `update_device(client: NautobotClient, id: str, **updates) -> DeviceSummary`
     - Gets existing record, applies updates, calls `.save()`
   - `delete_device(client: NautobotClient, id: str) -> bool`
     - Gets record, calls `.delete()`, returns True
   - All operations wrap pynautobot errors via `client._handle_api_error()`
  </action>
