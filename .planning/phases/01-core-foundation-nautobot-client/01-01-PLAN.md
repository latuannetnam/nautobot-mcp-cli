1. `NautobotClient` class:
   - `__init__(self, profile: NautobotProfile = None, settings: NautobotSettings = None)` — if no profile, load from settings
   - `self._api: pynautobot.api` — initialized with url, token, retries=3, ssl_verify
   - Property `api` — lazy initialization of pynautobot.api
   - `validate_connection(self) -> dict` — check server reachability, return `{"status": "connected", "version": "2.x", "url": "..."}`

   - Property accessors for endpoints: `self.dcim`, `self.ipam`, `self.tenancy`, `self.circuits`
