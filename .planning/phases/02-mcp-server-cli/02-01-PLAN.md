Create `nautobot_mcp/server.py` with:

1. FastMCP server instance: `mcp = FastMCP("Nautobot MCP Server")`

2. Client factory function:
```python
from nautobot_mcp.client import NautobotClient
from nautobot_mcp.config import NautobotSettings

_client: NautobotClient | None = None

def get_client() -> NautobotClient:
    global _client
    if _client is None:
        settings = NautobotSettings()
        _client = NautobotClient(settings=settings)
    return _client
```

3. Error handler helper:
```python
from fastmcp.exceptions import ToolError
from nautobot_mcp.exceptions import NautobotMCPError

def handle_error(e: Exception) -> None:
    if isinstance(e, NautobotMCPError):
        msg = e.message
        if hasattr(e, 'hint') and e.hint:
            msg += f". Hint: {e.hint}"
        raise ToolError(msg)
    raise ToolError(f"Unexpected error: {str(e)}")
```

4. Device tools (5 tools):
- `nautobot_list_devices(location: str | None = None, tenant: str | None = None, role: str | None = None, platform: str | None = None, q: str | None = None, limit: int = 50) -> dict` — calls `list_devices(client, ...)`, returns `result.model_dump()`
- `nautobot_get_device(name: str | None = None, id: str | None = None) -> dict`

- `nautobot_update_device(id: str, **updates) -> dict`
- `nautobot_delete_device(id: str) -> dict` — returns `{"success": True, "message": "Device deleted"}`

Each tool has:
- `@mcp.tool(name="nautobot_...", description="Rich description with parameter docs")`

- Returns dict via `.model_dump()` on pydantic results
