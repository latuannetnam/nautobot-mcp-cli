<action>
Create `nautobot_mcp/cms/arp.py` with 2 read-only CRUD functions:

**1. `list_arp_entries(client, device=None, interface=None, mac_address=None, limit=0)`**
- If `device` provided: call `resolve_device_id()` then `cms_list("juniper_arp_entries", ArpEntrySummary, device=device_id, limit=limit)`
- If `interface` provided: pass as filter `interface=interface`

**2. `get_arp_entry(client, id)`**
- Call `cms_get("juniper_arp_entries", ArpEntrySummary, id=id)`
- Returns `ArpEntrySummary`
