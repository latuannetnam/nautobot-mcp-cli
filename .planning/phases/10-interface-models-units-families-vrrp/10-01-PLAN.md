# Plan 10-01: Interface Pydantic Models & Core CRUD Functions

<objective>
Create Pydantic models for all 7 interface CMS models and domain-specific CRUD functions in `nautobot_mcp/cms/interfaces.py` that use the Phase 8 generic CRUD helpers. Hybrid inlining: `list_interface_units` is shallow (includes `family_count`), `get_interface_unit` inlines families with filter/policer names. InterfaceUnit uses `device` filter for device-scoped queries. VLANs stored as UUID lists only. Filter/policer associations get create/delete. VRRP tracking is read-only.
</objective>

<action>
Create `nautobot_mcp/cms/interfaces.py` with these functions:

**Interface Units (full CRUD, device-scoped):**
- `list_interface_units(client, device, limit=0)` → `ListResponse[InterfaceUnitSummary]`
  - Resolves device name to UUID via `resolve_device_id`
  - Calls `cms_list(client, "juniper_interface_units", InterfaceUnitSummary, device=device_id)`
  - Shallow response: no inlined families, only `family_count` integer
  - To compute `family_count`: after fetching units, batch-fetch all families for device's units in one call using `interface_unit__in=[unit_ids]` if API supports, or compute client-side from a single families query
- `get_interface_unit(client, id)` → `InterfaceUnitSummary`
  - Rich response: inlines family summaries with filter/policer names
  - After getting unit, fetch families for that unit, then for each family fetch filters and policers
  - Add `families: list[InterfaceFamilySummary]` as an optional enrichment field (not in base model — either add to model or return as dict)
- `create_interface_unit(client, interface_id, vlan_mode=None, encapsulation=None, **kwargs)` → `InterfaceUnitSummary`
- `update_interface_unit(client, id, **updates)` → `InterfaceUnitSummary`
- `delete_interface_unit(client, id)` → `dict`

All functions follow the established pattern:
```python
def list_interface_units(client: NautobotClient, device: str, limit: int = 0) -> ListResponse[InterfaceUnitSummary]:
    try:
        device_id = resolve_device_id(client, device)
        result = cms_list(client, "juniper_interface_units", InterfaceUnitSummary, device=device_id, limit=limit)
        # compute family_count for each unit (batch fetch families)
        return result
    except Exception as e:
        client._handle_api_error(e, "list", "InterfaceUnit")
        raise
```
