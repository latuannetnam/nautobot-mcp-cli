<action>
1. Create `nautobot_mcp/cli/cms_interfaces.py` with Typer subcommand group.

2. Register it in `nautobot_mcp/cli/app.py`:
   - Create `interfaces_app = typer.Typer(name="interfaces", help="Juniper interface model operations")`
   - Add `cms_app.add_typer(interfaces_app)` (use existing `cms_app` created for routing)

3. CLI commands:

**Interface Units:**
- `list-units --device NAME [--limit 50] [--detail]`
  - Default table columns: Interface | Unit# | VLAN Mode | Encapsulation | QinQ | Families | Description
  - `--detail` adds: Outer VLANs, Inner VLANs, Router Tagged VLAN, GigE Speed, LACP
- `get-unit --id UUID`
  - Shows unit info + inlined families table (Type | MTU | Filters | Policers)
- `create-unit --interface-id UUID [--vlan-mode TEXT] [--encapsulation TEXT] [--description TEXT]`
- `delete-unit --id UUID [--confirm/--yes]`

**Interface Families:**
- `list-families --unit-id UUID [--limit 50]`
  - Table columns: Family Type | MTU | Filters | Policers
- `create-family --unit-id UUID --family-type TEXT`
- `delete-family --id UUID [--confirm/--yes]`

**Filter/Policer Associations:**
- `list-family-filters --family-id UUID [--limit 50]`
  - Table columns: Filter Name | Type | Enabled
- `create-family-filter --family-id UUID --filter-id UUID --filter-type TEXT [--enabled]`
- `delete-family-filter --id UUID [--confirm/--yes]`
- `list-family-policers --family-id UUID [--limit 50]`
  - Table columns: Policer Name | Type | Enabled
- `create-family-policer --family-id UUID --policer-id UUID --policer-type TEXT [--enabled]`
- `delete-family-policer --id UUID [--confirm/--yes]`

**VRRP Groups:**
- `list-vrrp-groups --family-id UUID [--limit 50]`
  - Table columns: Group# | Virtual IP | Priority | Preempt | Accept Data | Auth | Tracks
- `get-vrrp-group --id UUID [--detail]`
  - `--detail` shows tracked routes and tracked interfaces tables
- `create-vrrp-group --family-id UUID --group-number INT --virtual-address-id UUID [--priority 100]`
- `delete-vrrp-group --id UUID [--confirm/--yes]`

**VRRP Tracking (read-only):**
- `list-vrrp-track-routes --vrrp-group-id UUID [--limit 50]`
  - Table columns: Route Address | Priority Cost | Routing Instance
- `list-vrrp-track-interfaces --vrrp-group-id UUID [--limit 50]`
  - Table columns: Tracked Interface | Priority Cost

Each command uses `rich.table.Table` for tabular output and `rich.console.Console` for `--detail` output.
All commands use the pattern:
```python
@interfaces_app.command("list-units")
def list_units_cmd(device: str = typer.Option(..., help="Device name"), ...):
    """List Juniper interface units for a device."""
    client = get_client_from_config()
    result = cms_interfaces.list_interface_units(client, device=device, ...)
    # Format as rich table
```
</action>
