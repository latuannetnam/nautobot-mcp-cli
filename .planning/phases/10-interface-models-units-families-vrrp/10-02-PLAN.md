Each tool follows the exact pattern:
```python
@mcp.tool(name="nautobot_cms_list_interface_units")
def nautobot_cms_list_interface_units(
    device: str,
    limit: int = 50,
) -> dict:
    """List Juniper interface units for a device.
    Returns shallow results with family_count per unit.
    """
    try:
        client = get_client()
        result = cms_interfaces.list_interface_units(client, device=device, limit=limit)
        return result.model_dump()
    except Exception as e:
        handle_error(e)
```
