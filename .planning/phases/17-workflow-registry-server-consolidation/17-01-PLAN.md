<action>
Create `nautobot_mcp/workflows.py` with:

1. **Imports** of all 10 composite functions from their modules:
   - `from nautobot_mcp.cms.routing import get_device_bgp_summary` (also uses get_routing_table)
   - `from nautobot_mcp.cms.firewalls import get_device_firewall_summary`
   - `from nautobot_mcp.cms.interfaces import get_interface_detail`
   - `from nautobot_mcp.verification import verify_config_compliance, verify_data_model`
   - `from nautobot_mcp.onboarding import onboard_config`
   - `from nautobot_mcp.drift import compare_device`
   - `from nautobot_mcp.cms.cms_drift import compare_bgp_neighbors, compare_static_routes`
   - `from nautobot_mcp.parsers.models import ParsedConfig`
   - `from nautobot_mcp.catalog.workflow_stubs import WORKFLOW_STUBS`

2. **WORKFLOW_REGISTRY** dict mapping workflow_id → entry:
   ```python
   WORKFLOW_REGISTRY = {
       "bgp_summary": {
           "function": get_device_bgp_summary,
           "param_map": {"device": "device", "detail": "detail"},
           "required": ["device"],
       },
       "routing_table": {
           "function": get_routing_table,  # from cms.routing
           "param_map": {"device": "device", "routing_table": "routing_table"},
           "required": ["device"],
       },
       "firewall_summary": {
           "function": get_device_firewall_summary,
           "param_map": {"device": "device", "detail": "detail"},
           "required": ["device"],
       },
       "interface_detail": {
           "function": get_interface_detail,
           "param_map": {"device": "device", "include_arp": "include_arp"},
           "required": ["device"],
       },
       "onboard_config": {
           "function": onboard_config,
           "param_map": {"config_data": "parsed_config", "device_name": "device_name", "dry_run": "dry_run"},
           "required": ["config_data", "device_name"],
           "transforms": {"config_data": lambda d: ParsedConfig.model_validate(d)},
       },
       "compare_device": {
           "function": compare_device,
           "param_map": {"device_name": "device_name", "live_data": "interfaces_data"},
           "required": ["device_name", "live_data"],
       },
       "verify_data_model": {
           "function": verify_data_model,
           "param_map": {"device_name": "device_name"},
           "required": ["device_name"],
       },
       "verify_compliance": {
           "function": verify_config_compliance,
           "param_map": {"device_name": "device_name"},
           "required": ["device_name"],
       },
       "compare_bgp": {
           "function": compare_bgp_neighbors,
           "param_map": {"device_name": "device_name", "live_neighbors": "live_neighbors"},
           "required": ["device_name", "live_neighbors"],
       },
       "compare_routes": {
           "function": compare_static_routes,
           "param_map": {"device_name": "device_name", "live_routes": "live_routes"},
           "required": ["device_name", "live_routes"],
       },
   }
   ```

3. **`_serialize_result(obj)`** helper: converts Pydantic models (`.model_dump()`), dataclasses (`dataclasses.asdict()`), dicts (pass-through), and anything else (`str(obj)`).

4. **`_build_envelope(workflow_id, params, data, error)`** helper: returns
   ```python
   {
       "workflow": workflow_id,
       "device": params.get("device") or params.get("device_name"),
       "status": "error" if error else "ok",
       "data": data,
       "error": str(error) if error else None,
       "timestamp": datetime.now(timezone.utc).isoformat(),
   }
   ```

5. **`run_workflow(client, workflow_id, params)`** function:
   - Validate `workflow_id` exists in `WORKFLOW_REGISTRY`. If not, raise `NautobotValidationError` with hint listing available workflows.
   - Validate required params from registry entry.
   - Map agent-facing param names to function arg names using `param_map`.
   - Apply `transforms` if present (e.g., `ParsedConfig.model_validate()`).
   - Inject `client` as first positional arg.
   - Call the function.
   - Serialize result via `_serialize_result()`.
   - Return `_build_envelope()`.
</action>

<acceptance_criteria>
- File `nautobot_mcp/workflows.py` exists
- `WORKFLOW_REGISTRY` has exactly 10 entries
- `run_workflow` function has signature `(client, workflow_id: str, params: dict) -> dict`
- `_serialize_result` handles Pydantic BaseModel, dataclass, and dict
- `_build_envelope` returns dict with keys: workflow, device, status, data, error, timestamp
- Invalid workflow_id raises NautobotValidationError
- Missing required params raises NautobotValidationError
- `python -c "from nautobot_mcp.workflows import WORKFLOW_REGISTRY; print(len(WORKFLOW_REGISTRY))"` prints `10`
</acceptance_criteria>
