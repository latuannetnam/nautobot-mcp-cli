# Plan 04-02: Verification Engine & Drift Report

<objective>
Create the verification engine using DiffSync for object-by-object comparison between live router state (ParsedConfig) and Nautobot records. Produces structured drift reports grouped by type (interfaces, IPs, VLANs) with missing/extra/changed categories.
</objective>

- DiffSync dependency added to pyproject.toml
- DiffSyncModel subclasses for interfaces, IPs, VLANs

- Structured DriftReport model (grouped by type: missing, extra, changed)
- Config compliance using existing quick_diff_config for VERIFY-01
- Flexible input: accept pre-fetched config or try jmcp
- Report suitable for both human (table) and agent (JSON) output
</must_haves>

**DiffSync models:**

```python
from diffsync import Adapter, DiffSyncModel

class SyncInterface(DiffSyncModel):
    _modelname = "interface"
    _identifiers = ("device_name", "name",)
    _attributes = ("description", "enabled",)

    device_name: str
    name: str
    description: str = ""
    enabled: bool = True

class SyncIPAddress(DiffSyncModel):
    _modelname = "ipaddress"
    _identifiers = ("address",)
    _attributes = ("interface_name", "family",)

    address: str
    interface_name: str = ""
    family: str = "inet"

class SyncVLAN(DiffSyncModel):
    _modelname = "vlan"
    _identifiers = ("vlan_id",)
    _attributes = ("name", "description",)

    vlan_id: int
    name: str = ""
    description: str = ""
```

**Adapters:**

1. **ParsedConfigAdapter** — loads from ParsedConfig:
```python
class ParsedConfigAdapter(Adapter):
    interface = SyncInterface
    ipaddress = SyncIPAddress
    vlan = SyncVLAN
    top_level = ["interface", "ipaddress", "vlan"]

    def __init__(self, parsed_config: ParsedConfig, device_name: str, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.parsed_config = parsed_config
        self.device_name = device_name

    def load(self):
        for iface in self.parsed_config.interfaces:
            self.add(SyncInterface(
                device_name=self.device_name,
                name=iface.name,
                description=iface.description,
                enabled=iface.enabled,
            ))
            for unit in iface.units:
                for ip in unit.ip_addresses:
                    self.add(SyncIPAddress(
                        address=ip.address,
                        interface_name=f"{iface.name}.{unit.unit}",
                        family=ip.family,
                    ))
        for vlan in self.parsed_config.vlans:
            self.add(SyncVLAN(
                vlan_id=vlan.vlan_id,
                name=vlan.name,
                description=vlan.description,
            ))
```

2. **NautobotLiveAdapter** — loads from Nautobot API:
```python
class NautobotLiveAdapter(Adapter):
    interface = SyncInterface
    ipaddress = SyncIPAddress
    vlan = SyncVLAN
    top_level = ["interface", "ipaddress", "vlan"]

    def __init__(self, client: NautobotClient, device_name: str, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.client = client
        self.device_name = device_name

    def load(self):
        # Load interfaces from Nautobot
        ifaces = list_interfaces(self.client, device=self.device_name)
        for iface in ifaces.results:
            self.add(SyncInterface(
                device_name=self.device_name,
                name=iface.name,
                description=iface.description,
                enabled=iface.enabled,
            ))
        # Load IPs for the device
        ips = list_ip_addresses(self.client, device=self.device_name)
        for ip in ips.results:
            self.add(SyncIPAddress(
                address=ip.address,
                interface_name=ip.interface or "",
                family="inet" if "." in ip.address else "inet6",
            ))
        # Load VLANs (device-scoped if possible)
        vlans = list_vlans(self.client)
        for vlan in vlans.results:
            self.add(SyncVLAN(
                vlan_id=vlan.vid,
                name=vlan.name,
                description=vlan.description,
            ))
```

Add the main verification functions to `nautobot_mcp/verification.py`:

1. **`verify_config_compliance(client, device_name, live_config=None) -> DriftReport`**
   - If live_config provided: use it. Otherwise: note in report.
   - Call `quick_diff_config(client, device_name)` for config compliance
   - Return DriftReport with `config_compliance` field populated

2. **`verify_data_model(client, device_name, parsed_config) -> DriftReport`**
   - Create `ParsedConfigAdapter(parsed_config, device_name)` and `NautobotLiveAdapter(client, device_name)`
   - Call `.load()` on both adapters
   - Call `nautobot_adapter.diff_from(parsed_adapter)` to get diff
   - Translate DiffSync diff output into `DriftReport` model with `DriftSection`s

3. **`_diffsync_to_drift_report(diff, device_name) -> DriftReport`**
   - Helper to translate DiffSync Diff object into our DriftReport
   - Map DiffSync "+" (create) → "missing_in_nautobot"
   - Map DiffSync "-" (delete) → "missing_on_device"
   - Map DiffSync "~" (update) → "changed" with changed_fields detail

4. **`_build_summary(report) -> dict`**
   - Count total drifts across all sections
   - Return `{"total_drifts": N, "interfaces": {...}, "ip_addresses": {...}, "vlans": {...}}`
